# bean-mock-api-service
Mock API service


# Quick Start

## install
```bash
npm install
```

## start service
```bash
node index.js
```

## test API

### send request
```bash
curl "http://127.0.0.1:1492/api/heartbeat"
```

### response
```json
{
    "code": "0",
    "message": "ok"
}
```

## stop service
Press CTRL+C to stop the nodejs process.
